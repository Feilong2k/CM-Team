# Bug Report B-0002

## Title
Ordering of Basic Info Categories in Entity Modal

## Description
The basic info section in the Entity Modal does not display categories in the expected order. The current order does not match the intended structure outlined in the project documentation.

## Steps to Reproduce
1. Open the Entity Modal in the application.
2. Navigate to the Basic Info tab.
3. Observe the order of categories displayed.

## Expected Behavior
The categories in the Basic Info section should be displayed in the following order:
1. Vision & Scope
2. High-Level Architecture
3. Cross-Cutting Concerns
4. Open Questions
5. Summary

## Actual Behavior
The categories are displayed in a different order, which may lead to confusion and misinterpretation of the information presented.

## Impact
This issue affects the clarity and usability of the Entity Modal, making it difficult for users to quickly find relevant information.

## Suggested Next Steps
- Review the rendering logic for the Basic Info section in the Entity Modal.
- Implement a deterministic ordering for the categories based on the specified structure.
- Test the changes to ensure the correct order is displayed.
